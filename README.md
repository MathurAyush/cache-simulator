# Cache Hit/Miss Visualizer Simulator

An interactive web-based Cache Memory Simulator
built for Computer Organization and Architecture (COA).

## Live Demo
https://mathaurayush.github.io/cache-simulator/

## Features
- Direct, Set-Associative and Fully Associative mapping
- LRU and FIFO replacement policies
- Miss Type Detection: Compulsory, Conflict, Capacity
- FIFO vs LRU Comparison Mode
- Step-by-step simulation with animations
- Real-time Hit Ratio Chart
- AMAT formula with live values

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js
